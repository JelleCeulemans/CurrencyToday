Wat is het?

Crytocurrency of de letterlijke betekenis ‘Versleuteld Geld’ word gezien als een vorm van internetgeld. Het is eigenlijk vergelijkbaar met gewoon geld, alleen bestaat het niet in een fysieke vorm. Het is eigenlijk gewoon een getal in een database waar een bepaalde waarde aan gegeven word. Het kan ook net als gewoon geld gebruikt worden om betalingen te verrichten, digitaal natuurlijk.

Het grote verschil met geld dat wij kennen, de euro en doller, is dat bij een handeling met cryptocurrency dit gebeurd zonder tussenkomst van een derde partij zoals een bank of de overheid. Het geld verspreid zich via een peer-to-peer netwerk. Dit wil dus zeggen dat het rechtstreeks van gebruiker naar gebruiker verspreid word.

Wallet

Om je eigen cryptocurrencies te beheren heb je iets nodig om ze op te slaan en op te volgen. Dit kan je doen door gebruikt te maken van een wallet, een digitale natuurlijk. Een wallet aanmaken kan je vaak doen via dezelfde site waar je de cryptocurrencies kan kopen. Je wallet bevat een speciale en unieke code. Deze code gebruik je om transacties te doen, zowel om currencies zelf aan te kopen en om aankopen te doen met de cryptocurrencies. Deze code heb je nodig om, via de vele website waar je cryptocurrencies kan aankopen, voor een soort afleveradres te zorgen.

Indien je een aankoop wilt doen met een cryptocurrency, dan heb je de code nodig van de wallet van de andere persoon. Een wallet aanmaken kan je op verschillende manieren doen. Je kan dit doen via dezelfde sites waar je de cryptocurrencies kan kopen. De tweede manier is om een wallet lokaal op je eigen computer te beheren. Het voordeel van de tweede manier is dat het veiliger is, maar je moet je wallet wel volledig zelf beheren. Bij de eerste manier wordt je wallet gedeeltelijk door de online service beheerd.

Serienummer

Om nu precies te weten wat een cryptocurrency is en hoe het opgebouwd is moeten we iets meer in detail kijken. Een cryptocurrency bestaat eigenlijk uit een serienummer, hierin zit een code die alle details van de currency bevat. Gewoon geld heeft ook een serienummer alleen is dit nummer op het biljet te vinden. Bij een cryptocurrency is dit volledig anders.

Het serienummer bij een cryptocurrency is versleuteld en is helemaal digitaal verwerkt in een netwerk. Hierdoor is het mogelijk om via een netwerk deze currencies te versturen naar andere personen. Doordat je het geld kan versturen over een netwerk, kan dit heel snel gebeuren.
